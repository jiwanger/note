简介

TypeScript 是 JavaScript 类型的超集，它可以编译成纯 JavaScript。

TypeScript 是微软开发的一个开源的编程语言，通过在 JavaScript 的基础上添加静态类型定义构建而成。TypeScript 通过 TypeScript 编译器或 Babel 转译为 JavaScript 代码，可运行在任何浏览器，任何操作系统。

TypeScript 添加了很多尚未正式发布的 ECMAScript 新特性。

TypeScript 扩展了 JavaScript 的语法，所以任何现有的 JavaScript 程序可以运行在 TypeScript 环境中。TypeScript 是为大型应用的开发而设计，并且可以编译为 JavaScript。

TypeScript 编译器，名称叫 tsc, 是用可以被编译为可以被执行在任何 JavaScript 引擎中，在任何宿主 - 如浏览器 - 中的常规 JavaScript 的 TypeScript 写的。编译器包被绑定于一个可以执行编译器的脚本宿主。使用 Node.js 作为宿主的 Node.js 包同样可以获得。

TypeScript 安装

$ npm install -g typescript

TypeScript 生成 JavaScript 
$ tsc greeter.ts # 将会生成 greeter.js
