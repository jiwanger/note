display 测试

1. block -- 不设置宽度时，宽度为父元素宽度；独占一行；支持宽高；

2. inline-block -- 不设置宽度时，内容撑开宽度；非独占一行；支持宽高；代码换行被解析成空格；

3. inline -- 内容撑开宽度；非独占一行；不支持宽高；代码换行被解析成空格；

4. none -- 隐藏元素并脱离文档流；

float 测试

1. 指定一个盒子（元素）是否应该浮动。

2. 绝对定位的元素忽略float属性！

3. 一个浮动元素会尽量向左或向右移动，直到它的外边缘碰到包含框或另一个浮动框的边框为止。

4. 浮动元素之后的元素将围绕它, 浮动元素之前的元素将不会受到影响。

5. clear 元素用来指定不允许元素周围有浮动元素。
