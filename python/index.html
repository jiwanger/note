简介

Python 是一种面向对象的解释型计算机程序设计语言。

Python 是纯粹的自由软件， 源代码和解释器 CPython 遵循 GPL 协议。

Python 语法简洁清晰，特色之一是强制用空白符作为语句缩进。

Python 最具特色的就是使用缩进来表示代码块，不需要使用大括号 {} 。

Python 使用缩进对齐组织代码的执行，所有没有缩进的代码（非函数定义和类定义），都会在载入时自动执行，这些代码，都可以可以认为是 Python 的 main 函数。每个文件（模块）都可以任意写一些没有缩进的代码，并且在载入时自动执行，为了区分主执行文件还是被调用的文件，Python 引入了一个变量 __name__，当文件是被调用时，__name__ 的值为模块名，当文件被执行时，__name__为 __main__。

Python 具有丰富和强大的库。它常被昵称为胶水语言，能够把用其他语言制作的各种模块（尤其是 C/C++）很轻松地联结在一起。常见的一种应用情形是，使用 Python 快速生成程序的原型（有时甚至是程序的最终界面），然后对其中有特别要求的部分，用更合适的语言改写，比如 3D 游戏中的图形渲染模块，性能要求特别高，就可以用 C/C++ 重写，而后封装为 Python 可以调用的扩展类库。需要注意的是在您使用扩展类库时可能需要考虑平台问题，某些可能不提供跨平台的实现。

Python 在执行时，首先会将 .py 文件中的源代码编译成 Python 的字节码，然后再由 PVM 来执行这些编译好的字节码。这种机制的基本思想跟 Java, .NET 是一致的。然而，PVM 与 Java 或 .NET 的 VM 不同的是，Python 的 VM 是一种更高级的 VM。这里的高级并不是通常意义上的高级，不是说 Python 的 VM 比 Java 或 .NET 的功能更强大，而是说和 Java 或 .NET 相比，Python 的 VM 距离真实机器的距离更远。或者可以这么说，Python 的 VM 是一种抽象层次更高的 VM。

Python 标识符

在 Python 里，标识符有字母、数字、下划线组成。

在 Python 中，所有标识符可以包括英文、数字以及下划线（_），但不能以数字开头。

Python 中的标识符是区分大小写的。

以下划线开头的标识符是有特殊意义的:

  • 单下划线开头（_foo）代表不能直接访问的类属性，需通过类提供的接口进行访问，不能用 from xxx import * 而导入；

  • 双下划线开头的（__foo）代表类的私有成员；

  • 双下划线开头和结尾的（__foo__）代表 Python 里特殊方法专用的标识，如 __init__（） 代表类的构造函数。

Python 行和缩进

学习Python与其他语言最大的区别就是，Python的代码块不使用大括号（{}）来控制类，函数以及其他逻辑判断。python最具特色的就是用缩进来写模块。

缩进的空白数量是可变的，但是所有代码块语句必须包含相同的缩进空白数量，这个必须严格执行。

多行语句 -- Python语句中一般以新行作为为语句的结束符。 但是我们可以使用斜杠（ \）将一行的语句分为多行显示

语句中包含[], {} 或 () 括号就不需要使用多行连接符。

# 同一行显示多条语句, 用分号缩进

# 将一行的语句分为多行显示

        item_two + \
        item_three

# [], {}, () 不需要使用多行连接符
days = ['Monday', 'Tuesday', 'Wednesday',
        'Thursday', 'Friday']

位运算符
运算符 描述 实例
& 按位与运算符：参与运算的两个值,如果两个相应位都为1,则该位的结果为1,否则为0 (a & b) 输出结果 12 ，二进制解释： 0000 1100
| 按位或运算符：只要对应的二个二进位有一个为1时，结果位就为1。 (a | b) 输出结果 61 ，二进制解释： 0011 1101
^ 按位异或运算符：当两对应的二进位相异时，结果为1 (a ^ b) 输出结果 49 ，二进制解释： 0011 0001
~ 按位取反运算符：对数据的每个二进制位取反,即把1变为0,把0变为1 (~a ) 输出结果 -61 ，二进制解释： 1100 0011， 在一个有符号二进制数的补码形式。
<< 左移动运算符：运算数的各二进位全部左移若干位，由"<<"右边的数指定移动的位数，高位丢弃，低位补0。 a << 2 输出结果 240 ，二进制解释： 1111 0000
>> 右移动运算符：把">>"左边的运算数的各二进位全部右移若干位，">>"右边的数指定移动的位数 a >> 2 输出结果 15 ，二进制解释： 0000 1111

内置函数

  1. print(*objects, sep=' ', end='\n', file=sys.stdout, flush=False) -- 将 objects 打印到 file 指定的文本流(默认标准输出，不自动刷新)，以 sep 分隔并在末尾加上 end

    print('Hello')
print(3, 'hello')
print(3, 'hello', sep=', ', end='==\n')

  2. bin(x) -- 将一个整数转变为一个前缀为 0b 开头的二进制字符串

  3. hex(x) -- 将一个将整数转换为以 0x 为前缀的小写十六进制字符串

  4. oct(x) -- 将一个整数转变为一个前缀为 0o 的八进制字符串

    a = 1024
print(bin(a), hex(a), oct(a))    # 0b10000000000 0x400 0o2000

  5. chr(i) -- 返回 Unicode 码位为整数 i 的字符的字符串格式

  6. ord(c) -- 对表示单个 Unicode 字符的字符串，返回代表它 Unicode 码点的整数

    u = 0x9648
print(chr(u), ord('陈'), hex(ord('陈')))  # 陈 38472 0x9648

  7. repr(object) -- 返回包含一个对象的可打印表示形式的字符串

  8. ascii(object) -- 就像函数 repr()，返回一个对象可打印的字符串，但是 repr() 返回的字符串中非 ASCII 编码的字符，会使用 \x、\u 和 \U 来转义。

    a = 1024
b = 0x400
c = '陈'
print(repr(a), repr(b), repr(c))      # 1024 1024 '陈'
print(ascii(a), ascii(b), ascii(c))   # 1024 1024 '\u9648'

  9. abs(x) -- 返回一个数的绝对值。实参可以是整数或浮点数。如果实参是一个复数，返回它的模。

Python 特殊方法

Python 以下划线开头的标识符是有特殊意义的:

  • 单下划线开头（_foo）代表不能直接访问的类属性，需通过类提供的接口进行访问，不能用 from xxx import * 而导入；

  • 双下划线开头的（__foo）代表类的私有成员；

  • 双下划线开头和结尾的（__foo__）代表 Python 里特殊方法专用的标识，如 __init__（） 代表类的构造函数。

def __or__(self, other): return self.age | other.age

def __xor__(self, other): return self.age ^ other.age

  10. __lshift__() / __rshift__() -- 操作符重载(左移和右移)

    def __lshift__(self, other): return self.age << other

def __rshift__(self, other): return self.age >> other

  11. __add__() / __sub__() / __mul__() / __truediv__() / __mod__() -- 操作符重载(+, -, *, /, %, //)

    def __add__(self, other): return self.age + other.age

def __sub__(self, other): return self.age - other.age

def __mul__(self, other): return self.age * other.age

def __truediv__(self, other): return round(self.age / other.age, 1)

def __mod__(self, other): return self.age % other.age

def __floordiv__(self, other): return self.age // other.age  # 除法，结果取整数，直接丢弃小数部分

  12. __neg__() / __pos__() -- 操作符重载(一元操作符-, +, ~)

    def __neg__(self): return -self.age

def __pos__(self): return self.age

def __invert__(self): return ~self.age

  13. __pow__() -- 乘方运算，pow() 函数和 ** 操作符调用此方法，注意，这个函数只适用于左边是对象，右边是数字

    def __pow__(self, power, modulo=None): return self.age ** power

p1 = Person("jiwanger", 32)
print(p1 ** 2, pow(p1, 2))   # 1024 1024
# print(2 ** p2)             # 错误，右边必须是数字，否则调用 __rpow__() 方法

  14. __abs__() / __divmod__() / __round__ / __hash__() / __next__() -- 内置函数的abs(), divmod(), round(), hash(), next() 方法分别调用对应的方法

    def __abs__(self): return abs(self.age)

def __divmod__(self, other): return divmod(self.age, other.age)

def __round__(self, n=None): return round(self.age, n)

def __hash__(self): return object.__hash__(self)

def __next__(self): return self.age   # 一般用于可迭代对象返回下一项

  15. __index__() -- 必须返回一个整数，此方法和 __int__() 有点类似，内置函数 bin() / hex() / oct() 取对象的 __index__() 返回值作为它们的参数

    def __index__(self): return self.age

  16. __int__() -- 必须返回一个整数，在 int 类型转换时调用，另外内置函数 chr() 取对象的 __int__() 返回值作为它的参数

    def __int__(self): return self.age

  17. __float__() / __complex__ -- 在类型转换时候，分别调用相应的方法

    def __int__(self): return self.age                        # 转换为 int 时

def __float__(self): return self.score                    # 转换为 float 时

def __complex__(self): return complex(self.age, self.age) # 转换为 complex 时

  18. __repr__() -- 返回包含一个对象的可打印表示形式的字符串，可以返回 __str__()

    def __repr__(self): return self.__str__()

  19. __ceil__ / __floor__ -- math 模块的 ceil() / floor() 方法取对象的对应方法的返回值为结果

    def __ceil__(self): return self.score

def __floor__(self): return self.score

import math

print(math.ceil(p1), math.ceil(p2))
print(math.floor(p1), math.floor(p2))

  20. __sizeof__() -- 返回对象的内存大小，sys.getsizeof(obj) 将会调用此方法计算对象的内存大小

    def __sizeof__(self):
	""" sys.getsizeof() 方法返回内置对象的内存大小，自定义的对象需要重写 __sizeof__() 自己实现 """
	return sys.getsizeof(self.name) + sys.getsizeof(self.age) + sys.getsizeof(self.score)
