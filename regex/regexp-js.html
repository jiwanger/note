RegExp 对象 RegExp 对象方法 支持正则表达式的 String 对象的方法

RegExp 对象

JavaScript提供了一个RegExp对象来完成有关正则表达式的操作和功能，每一条正则表达式模式对应一个RegExp实例。有两种方式可以创建RegExp对象的实例。

var pattern = new RegExp(pattern, modifiers);
// 或者更简单的方式:
var pattern = /pattern/modifiers;
	// i 	执行对大小写不敏感的匹配。
	// g 	执行全局匹配（查找所有匹配而非在找到第一个匹配后停止）。
	// m 	执行多行匹配。
  • pattern（模式） 描述了表达式的模式 -- 模式参考: 正则表达式元数据
  • modifiers(修饰符) 用于指定全局匹配、区分大小写的匹配和多行匹配

RegExp 对象方法

  • test() -- 检索字符串中指定的值。返回 true 或 false 

    var pattern = /hello/g;	// 在字符串中全局搜索 "hello" 
var str = "hello world!";
alert(pattern.test(str));

  • exec() -- 检索字符串中指定的值。返回找到的值，并确定其位置 

    var pattern = /is/g;
var str = "Age is 27, Year is 2012!";
var result = "";
while((result = pattern.exec(str)) != null) {
	console.log("匹配的数据: " + result + ", 位置: " + pattern.lastIndex);
}

  • compile() -- 编译正则表达式, 改变正则表达式模式 

    var pattern = /IS/g;
var str = "Age is 27, Year is 2012!";
console.log(pattern.test(str));
pattern.compile("is", "gi");// 改变模式
console.log(pattern.test(str));

支持正则表达式的 String 对象的方法

  • search() -- 检索与正则表达式相匹配的值。 

    var str = "Visit W3Cshool!";
// 找不到返回 -1; 找到返回起始位置
console.log(str.search("W3Cshool"));

  • match -- 找到一个或多个正则表达式的匹配。 

    var pattern = /IS/gi;
var str = "Age is 27, Year is 2012!";
var results = str.match(pattern);
for (var i = 0; i < results.length; i++) {
	console.log(i + ": " + results[i]);
}

  • replace -- 替换与正则表达式匹配的子串。 

    var str="Mr Blue has a blue house and a blue car";
str = str.replace(/blue/gi, "red");
console.log(str);

  • split -- 把字符串分割为字符串数组。 

    var str="Mr Blue has 27 blue house 28 a blue car";
var result = str.split(/[0-9]{2}/);
console.log(result);
