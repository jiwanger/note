简介

PHP 手册：PHP 官方中文手册

PHP(PHP：Hypertext Preprocessor) 是一种通用开源脚本语言。语法吸收了 C 语言、Java 和 Perl 的特点，利于学习，使用广泛，主要适用于 Web 开发领域。PHP 独特的语法混合了 C、Java、Perl 以及 PHP 自创的语法。它可以比 CGI 或者 Perl 更快速地执行动态网页。用 PHP 做出的动态页面与其他的编程语言相比，PHP是将程序嵌入到 HTML（标准通用标记语言下的一个应用）文档中去执行，执行效率比完全生成 HTML 标记的 CGI 要高许多; PHP 还可以执行编译后代码，编译可以达到加密和优化代码运行，使代码运行更快。

在 Linux 中一般通过 Apache 来使用 PHP，Windows 中则通过 IIS 来使用 PHP。

基本语法

PHP 基本语法

  • PHP 文件的默认文件扩展名是 .php;

  • PHP 脚本以 <?php 开始，以 ?> 结束：

    <?php 
	// php 代码 
?>

/* 如果在 php.ini 中配置了 short_open_tag=On，那么支持短标签 */
<?
	// php 代码
?>

  • PHP 中的每个代码行都必须以分号结束。

  • PHP 支持单行和多行注释：

    <?php
	// 这是 PHP 单行注释
	
	/*
	这是 PHP 多行 注释
	*/
?>

  • 如果文件内容仅仅包含 PHP 代码，最好在文件末尾删除 PHP 结束标记。

    <?php
	echo "Hello world";
	// ... more code
	echo "Last statement";

	// 脚本至此结束，并无 PHP 结束标记

  • PHP 的输出 echo 和 print: echo 输出的速度比 print 快， echo 没有返回值，print 有返回值1。

    <?
    $x = 5;
    $y = 6;
    $sum = $x + $y;

    // echo 可以输出一个或多个字符串
	echo "$x + $y = $sum 
    ";
    echo $x, " + ",  $y, " = ", $sum, "
    ";
    // print 只能输出一个变量，并且有一个返回值 1
    $ret = print "$x + $y = $sum 
    ";

    echo $ret
?>

<p>使用 <?= ?> 可以替代 echo: </p>

<?= $x, " + ",  $y, " = ", $sum; ?>

数据类型

PHP 支持 10 种原始数据类型。

  • 四种标量类型: bool, int, float, string

  • 四种复合类型：array, object, callable, iterable

  • 两种特殊类型：resource, NULL

<?
	require 'util/Log.class.php';

	$bool = true;           // 不区分大小写
	$int = 10;              
	$float = 3.14;           
	$string = "Hello";      

	Log::log(gettype($bool), gettype($int), gettype($float), gettype($string));
	Log::line();
	Log::log(is_bool($bool), is_int($int), is_integer($int), is_float($int), is_float($float), 
			is_double($float), is_string($string));

	$array = array();
	$object = new Log();

	Log::log(gettype($array), gettype($object));
?>
